Presentación web interactiva y profesional del elemento químico Helio (He), desarrollada con HTML, CSS y JavaScript vanilla.

## 🌟 Características

- **Diseño Profesional**: Interfaz moderna con tema oscuro y animaciones suaves
- **Totalmente Interactivo**: Tarjetas clickeables con información detallada
- **Modelo Atómico Animado**: Visualización del átomo de Helio con electrones en órbita
- **Responsive Design**: Adaptable a dispositivos móviles, tablets y desktop
- **Información Completa**: Propiedades, historia, aplicaciones y valor comercial
- **Sin Dependencias**: Código puro sin frameworks ni librerías externas

## 📋 Contenido

### Información Incluida

- ⚛️ Símbolo y número atómico
- ⚖️ Masa atómica
- 📅 Fecha de descubrimiento (1868)
- 👨‍🔬 Descubridores (Pierre Janssen y Norman Lockyer)
- 💰 Precio comercial actual
- 🌡️ Propiedades físicas y químicas
- 🔬 Aplicaciones industriales y científicas

### Aplicaciones Detalladas

1. **Medicina**: Resonancia magnética (MRI)
2. **Aeroespacial**: Presurización de cohetes
3. **Buceo**: Mezclas Heliox para inmersiones profundas
4. **Industria**: Soldadura, detección de fugas, semiconductores
5. **Investigación**: Criogenia y física de partículas
6. **Aerostática**: Globos meteorológicos y dirigibles

## 💻 Instalación y Uso

### Opción 1: Clonar el repositorio

```bash
git clone https://github.com/tu-usuario/helio-elemento.git
cd helio-elemento
```

Abre `index.html` en tu navegador favorito.

### Opción 2: Descarga directa

1. Descarga el repositorio como ZIP
2. Extrae los archivos
3. Abre `index.html` en tu navegador
